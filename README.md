# Incremental-Mass-Rewritten-save-converter
This is a tiny tool that converts the save file of v0.8 beta version of the game "Incremental Mass Rewritten" to v0.7.1.6.

Technically, you would only want to use this in the case that you switched to v0.8 before completing challenge 20 in the v0.7.1.6 version of the game and lost the v0.7.1.6 save. But this works too when you are in the evolution stage in the 0.8 version, although it would be meaningless to switch back to v0.7.1.6 version again at this point.
